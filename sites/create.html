Discord.bot

How to insatall files:

  1. First, create a folder where your bot will be stored
  2. Open in Visual Code and open the folder where your bot will be located
  3. Open a terminal
  4. Enter the command npm init -y, which will create a file in which information about the project will be stored
  5. After that, in the terminal write npm install discord.js
  6. Create a file index.js in the folder and you're done!
