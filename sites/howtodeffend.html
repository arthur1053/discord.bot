How to protekt your TOKEN?

If the project is posted on GitHub (for example, as for a portfolio), then your TOKEN may be visible to others. how to prevent this?

  1. Create a new file ".env".It's the file that will ignored by Github
  2. In this file file write TOKEN="(write your token here)"
  3. Than comeback into index.js
  4. Put in the start require('dotenv').config();
  5. And put client.login(process.env.TOKEN)
  6. Now you can relax and your TOKEN in save
Back Finish!
